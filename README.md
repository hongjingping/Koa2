# Koa2
全栈Koa2
###### Mac下面更新node版本
- `node -v`
- `sudo npm install -g n`
- `sudo n stable`
###### Mac下面上传github
- `git init`
-  `git remote add origin git@github.com:hongjingping/Koa2.git`
- `git status`
- `git pull origin master`
- `git push origin master`

###### Mac下面安装koa
- `sudo npm init -y`
- `sudo cnpm install --save koa`
